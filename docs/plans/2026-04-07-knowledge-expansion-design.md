# Knowledge Constellation Expansion Design

**Date:** 2026-04-07

## Goal

기존 볼트를 단순 개요 모음에서 벗어나, `연결점 발견`에 최적화된 연구형 지도로 재구성한다. 기존 문서는 출처와 인접 연결을 보강하고, 동시에 브리지 문서와 인접 분야 허브를 추가해 사용자가 스스로 떠올리지 못했던 사유 경로를 더 많이 보이게 만든다.

## Problem

현재 볼트는 주제 선정 자체는 강하지만, 문서 다수가 다음 한계를 가진다.

- 개별 노트가 개요 수준에 머물러 있다.
- `대표 텍스트와 후속 읽기`가 실제 웹 링크와 연구 단서로 연결되지 않는다.
- 브리지 문서가 부족해 서로 먼 폴더 사이의 숨은 연결이 충분히 드러나지 않는다.
- 인접 분야로 자연스럽게 번져 가는 탐색 경로가 약하다.

## Chosen Direction

승인된 방향은 `기존 기본셋 전수 보강 + 브리지 문서 확장 + 인접 분야 허브 추가`다.

- 기존 문서는 핵심 주장, 긴장점, 연결점, 실제 출처를 더 분명히 만든다.
- 새 브리지 문서는 여러 폴더를 한 번에 엮는 질문 중심 노드로 쓴다.
- 새 허브는 현재 폴더 체계를 깨지 않고 1~2단계 인접 분야를 여는 방식으로 추가한다.

## Content Strategy

### Existing Notes

기존 주제 문서는 현재 문체를 유지하되, 최소한 다음 기능을 갖게 만든다.

- 실제 읽을 수 있는 출처 링크
- 추상적 요약이 아니라 논쟁점과 오해 가능성
- 새 브리지 문서와의 교차 연결
- “다음에 어디로 가야 하는가”가 보이는 읽기 흐름

### Bridge Notes

브리지 문서는 개별 인물·개념 소개보다 상위 질문에 집중한다. 1차 후보는 다음과 같다.

- `의식`
- `돈과 신뢰`
- `죽음의 해석학`
- `상징과 제도`
- `에이전트와 규율`
- `제국의 물류`
- `예측과 통치`
- `측정과 실재`
- `의례와 합의`
- `인지적 비용`

### Adjacent Fields

인접 분야 허브는 현재 볼트의 바깥 경계를 넓히는 역할을 맡는다. 1차 후보는 다음과 같다.

- `복잡계`
- `사이버네틱스`
- `과학철학`
- `인류학`
- `언어학`
- `정치신학`

각 허브는 독립 백과사전이 아니라 기존 노드를 증폭하는 “확장 포털”처럼 쓴다.

## Navigation Strategy

- `content/index.md`에서 새 허브와 새 브리지 문서를 전면 노출한다.
- `00_메타/핵심 연결축.md`를 재작성해 더 많은 축과 확장 경로를 보여 준다.
- `00_메타/노드 인벤토리.md`와 `00_메타/연구 질문.md`를 새 구조에 맞게 갱신한다.
- 기존 허브 `index.md`들에 인접 분야로 건너가는 링크를 추가한다.

## Source Strategy

- 과학·건강·뇌과학은 DOI, PubMed, Nature/Annual Reviews 같은 연구 중심 링크를 우선 사용한다.
- 철학·종교·역사·사상가는 SEP, Britannica, 원전 또는 기관 자료를 조합한다.
- 비트코인·AI·법행정은 공식 문서, 표준, 백서, 정책 문서를 우선 사용한다.
- 모든 링크를 완전한 학술 장비로 통일하기보다, “원전 + 신뢰 가능한 개설 + 후속 연구”의 3층 구조를 지향한다.

## Scope

- `content/index.md`
- `content/00_메타/*.md`
- 기존 주제 허브 `content/*/index.md`
- 기존 주제 문서 다수
- 신규 허브 폴더와 신규 브리지 문서

## Validation

- `npm run build`
- `git diff --stat`
- 생성된 링크와 위키링크의 구조적 일관성 점검
